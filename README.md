# Love Minnie Pictures 🌻

## 📖 给徐老师的使用指南 (User Manual)

亲爱的徐老师，您的**“星空下的向日葵画廊”**已经搭建好啦！
这里是为您准备的最简操作手册，只需 3 步，即可让爱的画作上线。

---

### 第一步：准备画作 🎨

1.  画好今天的画（电子版）。
2.  将图片文件改名为今天的日期，格式如下：
    - `20251206.jpg` (年月日) 或 `2025-12-06.png`
    - 推荐使用 `YYYYMMDD` 格式，不仅好记，电脑排序也方便。

### 第二步：一键上传 📤

1.  打开项目文件夹，找到 **`raw_images`** 文件夹。
2.  把您的图片**拖进去**。
3.  双击运行项目里的 **`update_gallery.py`** 脚本（或者在终端输入 `python3 update_gallery.py`）。
    - _若谷提示：脚本会自动把您的图片变成网页版，不仅体积小（只有几百 KB），而且打开速度飞快！_

### 第三步：发布到世界 🌍

1.  打开终端，输入以下“咒语”把变动推送到 GitHub：
    ```bash
    git add .
    git commit -m "Add new daily painting"
    git push
    ```
2.  等待 1-2 分钟，打开您的专属网址（稍后在 GitHub Settings 开启 Pages 后获得），Minnie 就能看到啦！

---

## ✨ 特色功能说明

- **📅 时光画廊**：
  - **今天之前的日子**：显示您画的精美画作。
  - **未来的日子**：显示为一颗未发芽的**向日葵种子** 🌱，点击会提示“由爱灌溉中”。
- **🎟️ 爱的展票**：
  - 点开任意一幅画，会弹出大图。
  - 点击底部的 **“保存为图片 (Save Ticket)”** 按钮，会生成一张像博物馆门票一样的精美卡片，发给 Minnie 更有仪式感！
- **🎵 燕姿时光机**：
  - 点击右下角的**黑色唱片机**，它会开始旋转并播放音乐（需放入 MP3 文件）。

---

## 🔧 技术备注 (Technical Notes)

- **数据**：所有信息存储在 `data.json`，无需数据库。
- **图片**：原图存 `raw_images` (不上传)，网页图存 `images` (WebP 格式)。
- **依赖**：需要安装 Python 和 Pillow 库 (`pip install Pillow`)。
