# 🎉 纪念日特效设计规范 (Anniversary Effects Design Spec)

> **文档版本**：v1.0  
> **最后更新**：2025-12-27  
> **维护人**：若谷 & 徐老师

---

## 📌 文档说明

本文档定义了 Love Minnie 网站在**特殊日期**触发的视觉特效。每个纪念日都有独立的设计方案，确保每次庆祝都独一无二。

**如何添加新纪念日**：直接在下方 [纪念日列表](#-纪念日列表) 添加一个新条目，按照模板填写即可。

---

## 🎨 设计原则

1. **精致感优先**：禁止使用简单 Emoji 作为粒子，所有图形使用 SVG 或 Canvas 绑制的高级向量图。
2. **克制与优雅**：粒子数量控制在合理范围（< 150），确保 60fps 流畅运行。
3. **情感共鸣**：每个日期的视觉风格应与其情感内涵匹配。
4. **可选跳过**：特效播放后提供淡出机制，不强制用户观看。

---

## 🏷️ 方案类型定义

### 方案 A：盛大庆典 (Grand Celebration)
- **触发时机**：用户进入网站瞬间，自动播放一次。
- **表现形式**：全屏粒子爆发 + 祝福文字浮现。
- **持续时间**：约 4-5 秒，之后自动淡出。
- **适用场景**：生日、情人节等需要强烈惊喜感的日子。

### 方案 B：沉浸氛围 (Immersive Atmosphere)
- **触发时机**：用户进入网站后，作为背景层持续存在。
- **表现形式**：稀疏的粒子缓慢飘落/漂浮，不遮挡内容。
- **持续时间**：全程持续，直到用户离开或切换日期。
- **适用场景**：结婚纪念日、新年等需要长久氛围感的日子。

### 🦋 9月22日 —— Minnie 生日

| 属性 | 值 |
|------|-----|
| **日期** | 09-22 |
| **方案类型** | A (盛大庆典) |
| **设计灵感** | 迪士尼城堡烟火 + 梦幻蝴蝶园 |

#### 视觉效果
1. **进场动画 (约 4 秒)**：
   - 屏幕底部爆发出**一束柔光**，如同舞台追光灯打开。
   - 从光芒中飞出**数十只精致的 3D 渐变蝴蝶**（SVG 绘制，带有半透明翅膀和微妙的虹彩光泽）。
   - 蝴蝶颜色：**薰衣草紫 → 樱花粉 → 薄荷绿** 马卡龙渐变。
   - 蝴蝶轨迹：随机但优雅地飞舞上升，带有真实的翅膀扇动动画。
2. **主文字**：
   - 屏幕中央浮现：`"Happy Birthday, My Minnie"`
   - 字体：Bodoni Moda，金色描边 + 柔光效果。
3. **次级粒子**：
   - 伴随蝴蝶飞舞，散落少量**金色星尘**（tiny sparkles）。

#### 色彩方案
```css
--birthday-primary-1: #E8B4E8; /* 淡紫 */
--birthday-primary-2: #FFC7D4; /* 樱花粉 */
--birthday-primary-3: #B8E8D2; /* 薄荷绿 */
--birthday-accent: #FFD700;    /* 金色星尘 */
--birthday-overlay: rgba(0, 0, 0, 0.6); /* 背景遮罩 */
```

### 💕 12月10日 —— 恋爱纪念日

| 属性 | 值 |
|------|-----|
| **日期** | 12-10 |
| **方案类型** | A (盛大庆典) |
| **设计灵感** | 复古胶片电影 + 巴黎铁塔下的玫瑰 |

#### 视觉效果
1. **进场动画 (约 5 秒)**：
   - 屏幕上方像拉开幕布一样，缓缓落下一道**金色光幕**。
   - 从光幕中飘落**精致的玫瑰金几何图形**：
     - **六芒星** (象征守护)
     - **双环交织** (象征两人羁绊)
     - **抽象心形轮廓** (细线条勾勒的优雅爱心，非 emoji)
   - 图形带有**微妙的旋转和闪烁**，像复古电影中的光斑。
2. **主文字**：
   - 屏幕中央浮现：`"16 Years of Love"` 或 `"Day XXXX · Still in Love"`
   - 字体：霞鹜文楷 + 金色渐变。
3. **次级效果**：
   - 屏幕边缘有轻微的**胶片颗粒感**（Grain Overlay）。
   - 背景微微泛出**暖黄色调**，如同黄昏的巴黎。

#### 色彩方案
```css
--love-primary-1: #D4A574; /* 玫瑰金 */
--love-primary-2: #F5DEB3; /* 小麦金 */
--love-primary-3: #C9A86C; /* 古铜金 */
--love-accent: #FFFAF0;    /* 象牙白光点 */
--love-overlay: sepia(20%) brightness(1.1); /* 暖色滤镜 */
```

### 💒 12月16日 —— 结婚纪念日

| 属性 | 值 |
|------|-----|
| **日期** | 12-16 |
| **方案类型** | B (沉浸氛围) |
| **设计灵感** | 烛光晚餐 + 深红天鹅绒 |

#### 视觉效果
1. **氛围层 (全程持续)**：
   - **不打断用户浏览**，作为背景层持续飘落。
   - **深红色玫瑰花瓣**：SVG 绘制的写实花瓣（带有自然的卷边和脉络纹理）。
   - 花瓣飘落速度**极慢**（约 8-12 秒落地），像电影中的慢镜头。
   - 花瓣数量稀疏（同时在屏幕上不超过 15 片）。
2. **光效增强**：
   - 网站原有的"氛围光场"色调**自动切换为暖红色**。
   - 鼠标光标的跟随光晕也变为**淡淡的玫瑰红**。
3. **交互彩蛋**：
   - 用户鼠标划过花瓣时，花瓣会被轻轻"吹走"。
   - 偶尔（5%概率）点击花瓣，弹出一行随机的结婚誓言/情话。

#### 色彩方案
```css
--wedding-petal-1: #8B0000; /* 深红 */
--wedding-petal-2: #A52A2A; /* 褐红 */
--wedding-petal-3: #800020; /* 勃艮第酒红 */
--wedding-glow: #FFD1DC;    /* 淡粉微光 */
--wedding-ambient: rgba(139, 0, 0, 0.15); /* 整体偏暖红 */
```

### 🎆 1月1日 —— 新年

| 属性 | 值 |
|------|-----|
| **日期** | 01-01 |
| **方案类型** | B (沉浸氛围) |
| **设计灵感** | 东京跨年烟火 + 流金星河 |

#### 视觉效果
1. **氛围层 (全程持续)**：
   - 屏幕**顶部**偶尔（每 3-5 秒）绽放一朵**微型烟花**。
   - 烟花颜色：**香槟金、玫瑰金、冰蓝**交替。
   - 爆炸后粒子缓缓飘落，像金色雪花。
   - 爆炸点随机分布在屏幕上方 1/3 区域。
2. **光效增强**：
   - 背景氛围光增加**金色调**。
   - 偶尔有"极光带"从屏幕一侧缓慢飘过。
3. **文字提示 (可选)**：
   - 首次打开时，屏幕中央短暂浮现：`"Happy New Year, My Love"` (3秒后自动淡出)。

#### 色彩方案
```css
--newyear-firework-1: #FFD700; /* 金 */
--newyear-firework-2: #E8B4D8; /* 玫瑰金 */
--newyear-firework-3: #87CEEB; /* 冰蓝 */
--newyear-ambient: rgba(255, 215, 0, 0.08); /* 金色暖光点缀 */
```

### 💝 2月14日 —— 情人节

| 属性 | 值 |
|------|-----|
| **日期** | 02-14 |
| **方案类型** | A (盛大庆典) |
| **设计灵感** | 霓虹东京 + 心跳心电图 |

#### 视觉效果
1. **进场动画 (约 4 秒)**：
   - 屏幕中央先出现一颗**跳动的心形轮廓**（细线条绘制，带有心电图式的跳动动画）。
   - "砰"的一声，心形**爆裂**成满屏的**霓虹爱心**。
   - 爱心样式：由**发光的霓虹线条**勾勒的心形，带有玻璃质感和光晕（非填充 emoji）。

2. **主文字**：
   - `"Be My Valentine, Forever"` (霓虹灯字体效果，带电流闪烁)

#### 色彩方案
```css
--valentine-neon-1: #FF1493; /* 深粉 */
--valentine-neon-2: #FF6B6B; /* 珊瑚红 */
--valentine-neon-3: #FF69B4; /* 热粉 */

```

## 🔧 技术实现备注

- **粒子引擎**：使用 `canvas-confetti` 库 (MIT License)
- **SVG 图形**：存放于 `/images/effects/` 目录
- **触发逻辑**：由 `script.js` 中的 `AnniversaryDispatcher` 函数管理
- **性能优化**：粒子数量 < 150，使用 `requestAnimationFrame`
- **无障碍**：尊重用户的 `prefers-reduced-motion` 设置
